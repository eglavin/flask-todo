## Setup

1. [Setup venv](#setup-venv)
1. [Activate venv](#activate-venv)
1. [Install requirements](#install-requirements)
1. [Run Once](#run-once)
1. [Run Debug](#run-debug)

### Setup venv

```sh
python3 -m venv .venv
```

### Activate venv

```sh
. .venv/bin/activate
```

### Install requirements

```sh
pip install -r requirements.txt
```

### Run Once

```sh
flask --app main run
```

### Run Debug

```sh
flask --app main run --debug
```
